# Файловий менеджер на Expo (Лабораторна робота №5)

Простий мобільний файловий менеджер, розроблений на Expo (React Native) в рамках лабораторної роботи. Дозволяє виконувати базові операції з файлами та папками в межах локальної директорії `AppData` застосунку.

## Функціональні можливості

* **Навігація:** Перегляд вмісту папок, перехід у вкладені папки та повернення назад. Відображення поточного шляху.
* **Створення:** Створення нових папок та текстових файлів (`.txt`) з початковим вмістом.
* **Читання:** Перегляд вмісту `.txt` файлів.
* **Редагування:** Модифікація та збереження змін у `.txt` файлах.
* **Видалення:** Видалення файлів та папок (з підтвердженням).
* **Деталі:** Перегляд інформації про файл/папку (тип, розмір, дата модифікації).
* **Статистика:** Відображення загального, вільного та зайнятого місця на диску пристрою.

## Технології

* [Expo](https://expo.dev/)
* [React Native](https://reactnative.dev/)
* [Expo File System (`expo-file-system`)](https://docs.expo.dev/versions/latest/sdk/filesystem/)

## Запуск проекту

1.  **Клонуйте репозиторій:**
    ```bash
    git clone <URL репозиторію>
    cd <папка проекту>
    ```
2.  **Встановіть залежності:**
    ```bash
    npm install
    # або yarn install
    ```
3.  **Запустіть проект:**
    ```bash
    npx expo start
    ```
    Далі відскануйте QR-код за допомогою програми Expo Go на вашому мобільному пристрої або запустіть на емуляторі/симуляторі.

## Робоча директорія

Всі файлові операції виконуються виключно в межах директорії:
`[FileSystem.documentDirectory]/AppData/`

Ця директорія створюється автоматично при першому запуску програми, якщо вона відсутня.
