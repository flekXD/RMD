# Лабораторна робота №1

Простий мобільний додаток, створений за допомогою React Native та Expo, що демонструє використання верхньої навігації (Top Tabs).

## Опис

Цей додаток є першою лабораторною роботою і включає три основні екрани, навігація між якими здійснюється за допомогою вкладок у верхній частині екрану:

1.  **Головна (Новини):** Відображає список новин з текстом-заглушкою та місцем для зображення.
2.  **Фотогалерея:** Показує сітку фотографій (використовуються заглушки).
3.  **Профіль (Реєстрація):** Містить форму для введення даних користувача (електронна пошта, пароль, ПІБ).

Додаток також має заголовок з логотипом (заглушкою) та назвою "FirstMobileApp", а також футер з інформацією про автора на кожному екрані.

## Основні функції

* Верхня навігація за допомогою вкладок (`@react-navigation/material-top-tabs`).
* Відображення списку за допомогою `FlatList` на екрані новин.
* Відображення сітки на екрані галереї.
* Форма з полями `TextInput` та кнопкою `TouchableOpacity` на екрані профілю.
* Використання іконок (`@expo/vector-icons/Ionicons`) для вкладок навігації.
* Структура з компонентами для заголовка (`LogoHeader`) та елементів списку (`NewsItem`).
* Базове стилізування за допомогою `StyleSheet`.

## Використані технології

* React Native
* Expo
* JavaScript (ES6+)
* React Navigation v6 (`@react-navigation/native`, `@react-navigation/material-top-tabs`)
* Expo Vector Icons (`@expo/vector-icons/Ionicons`)

## Встановлення та запуск

1.  **Клонуйте репозиторій:**
    ```bash
    git clone <URL_вашого_репозиторію>
    cd <назва_папки_проекту>
    ```
2.  **Встановіть залежності:**
    Переконайтеся, що у вас встановлено Node.js та npm/yarn.
    ```bash
    npm install
    # або
    yarn install
    ```
3.  **Запустіть проект:**
    ```bash
    npx expo start
    # або
    yarn start
    ```
4.  **Перегляд:**
    * Відскануйте QR-код за допомогою програми Expo Go на вашому мобільному пристрої (Android/iOS).
    * Або запустіть на симуляторі/емуляторі (натисніть `i` для iOS симулятора або `a` для Android емулятора у терміналі, де запущено Expo).
